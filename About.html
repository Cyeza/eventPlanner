e-event planning The success of each event program

Due to the problem arising in organizing and managing an event, an online system will be developed in order to assist planner in managing event activities. This project Event Planning System will act as a planner to help and guides event committee in managing an event successfully. Event communities is this committee which involved wedding planner and meetings and other parties that have experienced in organizing an event.This project concentrates on developing a portal that is practical and efficient in this modern day as a guide to plan and manage an event better.
